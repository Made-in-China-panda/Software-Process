2016级项目实训成果展示
《攻略说》-HTML5与移动互联网开发
项目简介
攻略说是一款能提供各种生活小建议，帮助用户解决烦恼，提高生活质量的APP。
攻略说是全民适用的生活引导者，旨在提高用户生活质量，让每一位用户成为对生活有准备的人。
APP大体功能分为五个模块：恋爱交友、饮食健康、外出旅游、影视推荐、个人主页。
在社会竞争日益激烈的大环境下都市人生活节奏越来越快，往往会忽略人与人之间沟通需要的技巧，在攻略说中，无论是异性还是同性在这一模块总能找到适合你的交往技巧；对于社会亚健康问题的频繁出现，攻略说的饮食健康将成为用户的健康导师；外出旅游是生活中不可或缺的一部分，攻略说中有各种旅游景点介绍，无论是大都市还是小乡村总有适合你的出行选择；新时代各种电影电视蜂拥而至，攻略说则会为用户推荐各类高口碑优质大片。
